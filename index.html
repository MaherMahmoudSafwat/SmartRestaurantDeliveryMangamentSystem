⚠ Logical Errors Found & Fixed

  • DiscountStatus enum: Corrected to ACTIVE / EXPIRED / DISABLED / SCHEDULED.
  • Feedback.rating: Fixed from raw List to Integer (1–5).
  • Feedback.feedbackId: Removed self-referential FK.
  • OrderItem: Re-classified from "join table" to @Entity.
  • OrderLine: Renamed from ambiguous "Quantity".
  • IPaymentService: Added — was missing from both original files.
  • IReportService: Unified ITrackSales + ITotalSales.
  • PricingModel & PortionBasedType: Removed (duplicates of ItemType).
  • User entity: Restored role + loyaltyPoints fields.
  • Image entity: Restored size + fileName fields.
